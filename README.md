# Flipkart Television Market Analysis using Web Scraping & EDA

## Project Overview

This project analyzes television products sold on Flipkart using web scraping and Exploratory Data Analysis (EDA) to understand pricing patterns, customer preferences, brand performance, and key product features.

## Domain

Online Retail / E-commerce (Flipkart – Television Category)

## Objective

* Understand customer buying behavior
* Analyze the impact of price, discounts, brand, and features on ratings and demand
* Identify trends to support better pricing, inventory, and marketing strategies

## Data Collection

* Web scraping from Flipkart television listings
* Tools: Python, Requests, BeautifulSoup, Pandas

## Dataset Features

* Brand, MRP, Discount Price, Discount
* Ratings, Reviews
* Screen Size, Edition, Sound Output, Warranty

## Analysis Performed

* Data cleaning and preprocessing
* Univariate, Bivariate, and Multivariate analysis
* Visualization-based insight generation

## Key Insights

* Mid-range TVs (32–43 inches) dominate demand
* Brands like Samsung, LG, Sony, and Mi perform strongly
* Moderate discounts attract higher customer engagement
* Premium brands offer longer warranties and higher pricing

## Conclusion

The analysis highlights how pricing, brand, and product features influence customer satisfaction and demand, helping optimize business decisions in the e-commerce TV market.

## Tools Used

Python | Pandas | NumPy | Matplotlib | Seaborn | BeautifulSoup | Jupyter Notebook
